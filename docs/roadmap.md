# Roadmap

## MVP

- Tenant-aware dashboard.
- Script generation into short scenes.
- Asset library.
- Provider settings with encrypted API keys.
- FFmpeg render queue.
- Local and Docker deployment.

## Phase 2

- OpenAI/Anthropic/Gemini script provider selection.
- Pexels and paid stock media search.
- ElevenLabs voiceover generation.
- Cloudflare R2/S3 storage driver.
- Email notifications and webhook callbacks.

## Phase 3

- Social distribution to YouTube, Instagram, TikTok, LinkedIn, and Facebook.
- Provider failover and retry policies.
- Series planning with brand kits.
- Quotas, billing, SSO, MFA, and full SaaS tenant administration.
- Optional schema-per-tenant isolation for enterprise customers.
